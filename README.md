# **Реализация алгоритма CYK**

## 1. Сборка проекта

Сборка не требуется т.к. питон интерпретируемый. Для тестирования нужен pytest

```bash
pip install pytest
```

## 2. Запуск кода в режиме тестирования преподавателем

Программа считывает данные из stdin и выводит результат в stdout. Запуск:

```bash
python3 main.py
```

## 3. Запуск автоматизированных тестов на pytest

Есть готовые тесты, 2 грамматики, и по 5 слов для проверки на каждую. Запуск:

```bash
pytest -v test.py
```
